# إعداد سير عمل السندات

## موقع الإعداد
- **الملف:** `backend/models.py` (السطر 815)
- **الحقل:** `voucher_auto_post` في Settings model
- **API:** 
  - عرض: `GET /api/settings`
  - تحديث: `PUT /api/settings`

## الخيارات المتاحة

| الإعداد | القيمة | السلوك |
|---------|--------|---------|
| **الاعتماد قبل الترحيل** (افتراضي) | `false` | حفظ → اعتماد → ترحيل |
| **الترحيل التلقائي** | `true` | حفظ = اعتماد + ترحيل فوراً |

## تحديث الإعداد

```bash
PUT /api/settings
{
  "voucher_auto_post": true  # أو false
}
```

## الاستخدام الموصى به

- **الشركات الصغيرة:** `true` (ترحيل فوري)
- **الشركات المتوسطة/الكبيرة:** `false` (رقابة داخلية)
